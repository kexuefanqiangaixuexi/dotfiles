# dotfiles

Personal shell configuration and tooling preferences.

## What's included

| File | Purpose |
|------|--------|
| `.bashrc` | Bash configuration with prompt, history, tool integrations |
| `.aliases` | Git, Docker, Python shortcuts |
| `.gitconfig` | Git user config and aliases |
| `.tmux.conf` | Tmux with mouse support and key bindings |
| `.inputrc` | Readline vi-mode and history search |
| `terminal/settings.json` | VS Code editor preferences |

## Install

```bash
cd dotfiles
make install
```

## Update

```bash
make update
```
